Application web complète pour la gestion d'un cheptel d'élevage, permettant le suivi des animaux, la gestion des ventes et la génération de rapports détaillés. 
L'application est conçue pour faciliter la traçabilité des animaux et optimiser la gestion quotidienne d'une exploitation agricole.

TTechnologies utilisées pour ce mini projet

Frontend (React + TypeScript)

Backend (Node.js + Express + MySQL utilisant du php native comme server apache nous avons utiliser Xampp server pour notre cas)
